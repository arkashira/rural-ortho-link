# Rural Ortho Link

This project implements a secure video conferencing system for rural orthopedic care.

## Features

* Secure video conferencing with end-to-end encryption
* HIPAA compliance documentation
* Support for multiple participants

## Requirements

* Python 3.8 or later

## Installation

1. Clone the repository
2. Install dependencies using `poetry install`
3. Run tests using `pytest`

## Usage

1. Create a new instance of the `Teleconsultation` class
2. Add participants using the `add_participant` method
3. Start video conferencing using the `start_video_conferencing` method
4. Verify end-to-end encryption using the `verify_end_to_end_encryption` method
5. Document HIPAA compliance using the `document_hipaa_compliance` method
