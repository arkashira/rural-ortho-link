# Video Conferencing
A simple video conferencing system with end-to-end encryption and HIPAA compliance.

## Features
* Secure video conferencing with 2+ participants
* End-to-end encryption
* HIPAA compliance

## Usage
1. Create a `VideoConferencing` object
2. Add participants using the `add_participant` method
3. Start the conference using the `start_conference` method
4. Verify encryption and HIPAA compliance using the `verify_encryption` and `verify_hipaa_compliance` methods
